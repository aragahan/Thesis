HOW TO USE

Diabetes is a common disease spread throughout the Philippines and the entire world, and risk prediction of diabetes has been an emerging problem among patients. This web application aims to predict the risk of getting diabetes given a set of features or inputs through a machine learning algorithm.
Users need to input information in order for the web application to predict their risk of getting diabetes. ALL inputs need to be numbers, as the form will not accept words as inputs. ALL inputs need to be integer except for your input for Body Temperature, which accepts decimal numbers. Once the user fills up all the inputs correctly, press the PREDICT button and let the application run.
Alternatively, users can opt to upload a comma-separated values (CSV) file that contains all of the required features instead of manually inputting them. The CSV file must only contain one line of values.
